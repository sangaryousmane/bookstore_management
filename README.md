# Online Bookstore Management
This is an online bookstore management system that allows users to browse, search, and purchase books online. The system is built using Java and MySQL, and it provides a user-friendly interface for managing books, orders, and customers.

# Features
- Admin dashboard for overall management
- User authentication and authorization
- Book management (add, edit, delete)
- Order management (view, update, delete)
- Customer management (view, edit)
- Search functionality
- Shopping cart functionality
- Payment processing (via PayPal)

Installation
To run the application, you will need to have Java and MySQL installed on your system. You can download and install them from the following links:
- Java: https://www.java.com/en/download/
- MySQL: https://dev.mysql.com/downloads/

Once you have installed Java and MySQL, you can follow these steps to run the application:
1. Clone the repository to your local machine using the following command:

`git clone https://github.com/sangaryousmane/bookstore_management.git `

2. Open the project in your preferred Java IDE (such as Eclipse or IntelliJ IDEA).
3. Create a new MySQL database and import the **book_store**  database folder of the project.
4. Update the **application.properties** file located in the src/main/resources folder with your MySQL database credentials.
5. Run the application by executing the BookstoreManagementApplication.java file located in the src/main/java/com/bookstore folder.
6. Open your web browser and navigate to **http://localhost:8080** to access the application
